Ajouter les moments clés de votre vidéo

Nous avons une fonctionnalitée qui vous permet d'ajouter les moments clés de votre vidéo mais celle-ci est disponible uniquement sur tablette et ordinateurs.

Après avoir rempli tous les champs de votre partie vidéo cliquer sur générer vos données structurées.

Les moments clés sont aussi appelés "temps fort" ou "key moments", ce sont les moments importants de vos vidéos le chapitre.
Dans le code votre temps fort commencera à 00: 00: 00 secondes pour se conformer aux bonnes pratiques Google.
